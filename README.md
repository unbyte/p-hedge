# p-hedge

Request hedging for promises — launch redundant attempts with a delay and take the first to succeed. See [gRPC request hedging](https://grpc.io/docs/guides/request-hedging/) for the concept.

See [packages/p-hedge](./packages/p-hedge) for full documentation.

This repo also includes [p-dynamic](./packages/p-dynamic) — dynamic `Promise.any` / `Promise.race` that `p-hedge` builds on.
